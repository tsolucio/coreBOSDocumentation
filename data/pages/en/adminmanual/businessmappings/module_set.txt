====== Module Set Mapping ======

The purpose of this mapping is to define a heterogeneous set of module names. Simply a bunch of modules. This could be used to define a set of modules to launch a mass operation upon or a set of modules that must be excluded from some global process.

The accepted format is:
<code xml>
<map>
  <modules>
    <module>ModuleName</module>
    <module>ModuleName2</module>
    ...
  </modules> 
</map>
</code>

You will be able to get the set of modules using the getFullModuleSet() method.
